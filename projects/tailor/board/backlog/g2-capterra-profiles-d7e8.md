## Notes
Create product listings on G2 and Capterra. These platforms rank well for "[product] reviews" and "[product] alternatives" queries, provide dofollow backlinks, and build social proof for B2B/prosumer buyers.

## Checklist
- [ ] Create G2 vendor profile
- [ ] Create Capterra vendor profile
- [ ] Upload product screenshots, description, and pricing info
- [ ] Set up review collection flow (post-generation email prompt)
- [ ] Seed initial reviews from beta users
- [ ] Monitor and respond to reviews
